Shutter_places в замену Shutterstock Map Helper

После исчезновения из закромов Google Web Store расширения Shutterstock Map Helper для отображения детальной информации как: времени и географического положения покупателя картины многие задались вопросом: «Где замена?»

Что это такое?

Пользовательский скрипт shutter places служит для отображения тех мест, где были совершены последние покупки в Шаттре.

Установка расширения для запуска пользовательских скриптов

Чтобы устанавливать Userscripts (пользовательские скрипты) сначала нужно установить расширение в браузер (обычно, достаточно перейти по соответствующей ссылке):

  • TamperMonkey расширение для Mozilla Firefox
  • TamperMonkey расширение для Google Chrome
  • TamperMonkey расширение для Safari
  • TamperMonkey расширение для Opera

Часто задаваемые вопросы

У меня не получается установить, помогите!
Чтобы установить скрипт нужно сперва установить расширение Monkey, а затем просто перейти по ссылке. Предложенное всплываемое окно с предложением установить скрипт и будет нужным вам результатом. Пользуйтесь!
Скрипт я установил, а кнопка так и не появилась...
Попробуйте обновить свой браузер или сменить браузер на последнюю версию Google Chrome или Mozilla Firefox. Если всё установлено, попробуйте посетить защищенную версию сайта или незащищенную, соответственно (корневой сертификат, выдавший сертификат сайту Шаттра время от времени может истекать).
