DMT8 док-станция полицейских нательных камер

Полицейские камеры док-станция является последним зарядки и данных загрузки устройство, которое можно подключить до 8 камер, через порт USB 2.0, совместим с большинством полиции тела камер на рынке. В дополнение к общей мощности закрепления достигать 120 W, каждый порт может обеспечить до 2.0a самостоятельной адаптации тока, по словам подключенных устройств. Пользователям нужно только офисный компьютер, без обременительных проводов. К услугам гостей мульти устройство сбора и зарядки.

Особенности

  • Структура-цельнометаллический, это прочный и жесткий;
  • Полностью совместим с USB 1.1 и USB 2.0, поддержка горячей замены, совместимый с большинством тела носят фотокамеры.
  • С двойной защитой на выходе он может защищать подключенное устройство от перегрузки, поддержка перегрузки напряжения и мгновенной токовой защиты; Каждый порт обеспечивает до 2.0a текущей.
  • Правоохранительные рекордер клип может быть исправлено на восемь слотов, которые аккуратно организованы.
  • L-образный USB разъем, красивый и надежный;
  • Стыковка имеет четыре колодки могут стабильно размещены на любой плоской поверхности.
  • Док-станция имеет шесть отверстий, внутренняя системная плата оснащена вентилятором. Для предотвращения от простоя из-за перегрева.
  • Оборудован выключателем, который может быть непосредственно выключить восьми USB мощность.
  • С стены кронштейн может быть установлен на вертикальные стены.
  • Поддерживает Windows 7, Windows Vista, Windows XP Professional и Home Edition, Windows 98 / ME / 2000 профессиональных и другие
